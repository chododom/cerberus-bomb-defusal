Instructions

Rules

  • You must press the correct code to defuse the bomb
  • A wrong code will result in a 25% reduction of the time
  • If the countdown hits 00:00:00, the bomb explodes

Defusing

  1. Press the correct code, which consist out of 4 to 6 numbers and/or letters
  2. Press # to confirm
  3. If correct, the screen will show "DISARMED"
